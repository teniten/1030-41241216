Chart.js 官網

關於 Chart.js

Chart.js 是一個簡單且靈活的 JavaScript 圖表庫，能夠在 HTML5 Canvas 上繪製各種美觀的圖表，並提供豐富的自定義選項。

範例程式碼

<canvas id="myChart"></canvas>
<script>
    const ctx = document.getElementById('myChart').getContext('2d');
    new Chart(ctx, {
        type: 'bar',
        data: {
            labels: ['Red', 'Blue', 'Yellow', 'Green', 'Purple', 'Orange'],
            datasets: [{
                label: '# of Votes',
                data: [12, 19, 3, 5, 2, 3],
                backgroundColor: 'rgba(75, 192, 192, 0.2)',
                borderColor: 'rgba(75, 192, 192, 1)',
                borderWidth: 1
            }]
        }
    });
</script>
