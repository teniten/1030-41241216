D3.js 官網

關於 D3.js

D3.js 是一個強大的 JavaScript 函式庫，用於操作文件基於數據的文檔。它讓你能夠以數據驅動的方式將資料視覺化，支持許多不同的視覺化形式。

範例程式碼

<svg width="400" height="200"></svg>
<script>
    const data = [30, 80, 45, 60, 20, 90, 100];
    const svg = d3.select('svg');
    svg.selectAll('rect')
        .data(data)
        .enter().append('rect')
        .attr('x', (d, i) => i * 50)
        .attr('y', d => 200 - d)
        .attr('width', 40)
        .attr('height', d => d)
        .attr('fill', 'teal');
</script>
