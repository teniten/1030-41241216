Lottie 官網

關於 Lottie

Lottie 是一種用於展示動畫的輕量級文件格式，它使設計師能夠以 JSON 格式導出動畫，並在網頁或移動應用中流暢地使用這些動畫。

範例程式碼

<div id="lottie-animation"></div>
<script>
    lottie.loadAnimation({
        container: document.getElementById('lottie-animation'),
        renderer: 'svg',
        loop: true,
        autoplay: true,
        path: 'https://assets10.lottiefiles.com/packages/lf20_UJNc2t.json' // 替換為您的 JSON 動畫 URL
    });
</script>
