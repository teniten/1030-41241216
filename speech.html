關於 Web Speech API

Web Speech API 是一個可以讓網頁應用程序使用語音識別和語音合成功能的接口。這使得用戶可以用語音與網頁互動，提高了可訪問性和用戶體驗。

範例程式碼

<input type="text" id="text" placeholder="輸入文字">
<button onclick="speak()">播放語音</button>
