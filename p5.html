P5.js 介紹

P5.js 是一個 JavaScript 庫，旨在使創造性編程變得更加容易。它提供了一個簡單的 API，使開發者能夠創建互動式的圖形和動畫，並能夠輕鬆地處理音頻、影像和其他媒體。P5.js 特別適合藝術家、設計師和初學者，幫助他們快速實現創意。

範例程式碼


function setup() {
    createCanvas(400, 400);
}

function draw() {
    background(220);
    fill(150, 0, 0);
    ellipse(mouseX, mouseY, 50, 50);
}
